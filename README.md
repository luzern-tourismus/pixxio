# Pixxio

## Api Stable Version
https://api.pixxio.com/docs

## Api Unstable Version
https://api.pixxio.com/docs/unstable


## Installation
```
composer require luzerntourismus/pixxio
```

## Import Custommetadata
```
php bin/cmd.php pixxio-custommetadata-import
```

## Import Directory
```
php bin/cmd.php pixxio-directory-import
```

## Clean
```
php bin/cmd.php pixxio-clean
```
